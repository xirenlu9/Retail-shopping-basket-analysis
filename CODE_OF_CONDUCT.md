# 行为准则 (Contributor Covenant Code of Conduct)

## 我们的承诺

作为本项目的参与者、贡献者和维护者，我们致力于为每个人提供一个无骚扰、安全和包容的环境，无论性别、性取向、残疾、外貌、种族、年龄或宗教信仰如何。

## 我们的标准

参与本项目的正面行为包括：
*   使用包容性的语言
*   尊重不同的观点和经验
*   礼貌地接受建设性的批评
*   专注于对社区最好的事情
*   对其他社区成员表现出同理心

参与者不接受的行为包括：
*   使用性语言或图像，以及不受欢迎的性关注或调情
*   挑衅、侮辱/贬低评论以及人身或政治攻击
*   公开或私下的骚扰
*   在没有明确许可的情况下发布他人的私人信息，例如物理或电子地址
*   其他在专业环境中可能被合理认为是不当的行为

## 我们的责任

项目维护者有责任澄清和执行我们的可接受行为标准，并将对他们认为不当、威胁、冒犯或有害的行为采取适当和公平的纠正措施。

项目维护者有权且有责任删除、编辑或拒绝不符合本行为准则的评论、提交、代码、Wiki 编辑、问题和其他贡献，并可能暂时或永久禁止任何他们认为不当、威胁、冒犯或有害的其他行为的贡献者。

## 范围

本行为准则适用于项目空间内，也适用于个人在公共场所代表项目或其社区时。代表项目的示例包括使用官方项目电子邮件地址、通过官方社交媒体帐户发布，或在离线活动中担任指定代表。

## 执行

不当行为、骚扰或其他不可接受的行为可以通向项目维护者 [your-email@example.com] 举报。所有投诉都将被审查和调查，并将产生被认为是必要和适当的响应。项目团队将对举报人的信息保密。

维护者如果不诚实地遵守或执行行为准则，可能会面临由项目其他成员决定的临时或永久后果。

## 归属

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org)，版本 1.4，可在 [https://www.contributor-covenant.org/version/1/4/code-of-conduct.html](https://www.contributor-covenant.org/version/1/4/code-of-conduct.html) 查阅。
